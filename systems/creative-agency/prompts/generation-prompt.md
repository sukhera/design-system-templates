## Design Principles Used

1. **Gradient-First Design**
   - Gradients are the primary visual language
   - Multiple gradient presets for different use cases
   - Colored shadows that match gradient themes
   - Animated gradients for hero sections

2. **Bold Typography**
   - Display font for headlines (Cal Sans/DM Sans)
   - Larger type scale (up to 96px)
   - Gradient text for maximum impact
   - Generous line-height for readability

3. **Playful Interactions**
   - Smooth, bouncy animations
   - Hover scale effects
   - Gradient shifts on interaction
   - Glow effects on focus/hover

4. **Glassmorphism**
   - Backdrop blur effects
   - Semi-transparent surfaces
   - Works over gradient backgrounds
   - Border highlights

5. **Accessibility First**
   - High contrast text over gradients
   - Solid color fallbacks
   - Keyboard navigation support
   - Reduced motion preferences
   - WCAG AA compliance

## Customization Guide

### Changing Brand Colors

Update gradient presets in `tokens/colors.md` and `tokens/tailwind.config.js`:

```javascript
// Replace sunset gradient
backgroundImage: {
  'gradient-sunset': 'linear-gradient(135deg, YOUR_COLOR_1, YOUR_COLOR_2)',
}
```

### Adding New Gradients

1. Define in `tokens/colors.md`
2. Add to `tailwind.config.js`
3. Create matching colored shadow
4. Test contrast ratios

### Adjusting Typography

Update font stacks in `tokens/typography.md`:

```css
--font-display: 'Your Display Font', fallback;
--font-ui: 'Your UI Font', fallback;
```

### Modifying Spacing

Change base unit in `tokens/spacing.md`:

```javascript
// Switch from 8px to 4px base
spacing: {
  '1': '4px',
  '2': '8px',
  // ...
}
```
